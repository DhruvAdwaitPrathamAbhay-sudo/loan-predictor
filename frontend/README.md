# Loan Predictor Frontend

This is the React + Vite frontend for the Loan Approval Prediction System. It provides an interactive user interface for submitting loan applications and viewing real-time approval predictions.

## Quick Start

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

## Configuration
Make sure the FastAPI backend is running (typically on `http://localhost:8000`) before submitting predictions through the frontend dashboard. Adjust the API base URL in your frontend services/fetch logic if your backend is hosted elsewhere.
